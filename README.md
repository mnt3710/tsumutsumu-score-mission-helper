# ツムツム スコア計算ツール

Next.js (+ TypeScript) で作成した、ツムツムの「スコアの一の位を特定の数字にする」ための計算補助ツールです。

## 特徴
- 現在スコア判定（基礎スコアから最終スコアを計算し一の位を判定）
- 攻略方法提案（チェーン数 3〜5 に対するツムスコア候補を探索）
- 目標達成スコア一覧（基礎スコア一覧を昇順で表示）
- レベルボーナスはレベル範囲に応じて取得します
- 小数点含む計算は Math.floor で丸めています（ゲームの挙動に合わせる）

## 使い方
1. 依存関係をインストール: `npm install`
2. 開発サーバー起動: `npm run dev`
3. ブラウザで `http://localhost:3000` にアクセス

## 主要なフォルダ
- `app/` - Next.js App Router のページ
- `components/` - 各機能を分離したコンポーネント
- `utils/` - 計算ロジック（レベルボーナス、逆算など）

## 注意点
- 最終スコアは Math.floor(baseScore * (1 + bonusRate)) で算出します。
- レベルが範囲外の場合、ボーナス率は 0 として扱われます。

## テスト
- `npm run test` でユニットテストを実行します（Vitest）
